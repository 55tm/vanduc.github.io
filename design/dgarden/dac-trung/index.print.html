Sơ đồ trang web

Trang web sẽ tự động tạo tệp sitemap.xml, có sẵn /sitemap.xmltrên trang web của bạn. Điều này giúp các công cụ tìm kiếm lập chỉ mục trang web của bạn một cách chính xác, làm cho nó dễ được khám phá hơn. Để sơ đồ trang web được định dạng chính xác với URL đầy đủ cho tất cả các trang của bạn, plugin cần biết URL cơ sở của trang web của bạn. Bằng cách thêm URL của bạn vào cài đặt plugin, sơ đồ trang web sẽ tự động được tạo bằng giá trị đó.

Trang web sẽ tự động tạo nguồn cấp dữ liệu Atom/RSS có sẵn /feed.xmltrên trang web của bạn. Để tạo được điều này, bạn cần thêm URL cơ sở của mình vào cài đặt, như trong hình trên.
Để ngày hoạt động chính xác, bạn cần bật cài đặt Hiển thị dấu thời gian đã cập nhật trong cài đặt Giao diện của plugin. Nếu tính năng này không được bật, nguồn cấp dữ liệu sẽ sử dụng ngày cuối cùng mà trang web được xây dựng, trong thực tế có nghĩa là ngày cuối cùng bạn xuất bản bất kỳ nội dung nào lên trang web.
